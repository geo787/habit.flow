# Focus Flow+

AI-powered neuroadaptive productivity platform designed for people with ADHD and autism.

## Mission

Focus Flow+ helps neurodivergent users:
- manage focus
- reduce overwhelm
- break down tasks
- improve emotional regulation
- build adaptive routines

---

## Features

- Adaptive Focus Timer
- AI Task Breakdown
- Mood & Energy Tracking
- Emergency Reset Mode
- Analytics Dashboard
- Neurodivergent-Friendly UI

---

## Tech Stack

### Frontend

### Backend

- OpenAI API

## Vision

Focus Flow+ is more than a productivity app.

It is a neuroadaptive cognitive support system built to help neurodivergent individuals work in harmony with their own minds.
